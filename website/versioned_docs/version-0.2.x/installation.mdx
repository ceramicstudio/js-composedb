# Installation

:::caution Developer preview

ComposeDB packages are still under development and only released as a developer preview, they are **not ready for production use**.

There may be **breaking changes** between v0.x versions before a stable v1.0 version is released, causing models or composites created from previous versions not to be compatible.

:::

## Requirements

ComposeDB runs on top of [Ceramic](https://ceramic.network/). Running [a local Ceramic node](https://developers.ceramic.network/build/cli/installation/) is needed for app development.

## Development tools

ComposeDB provides libraries and a CLI to help support common development needs.

The CLI provides commands for the most common flows, while the libraries can be used in scripts to support more complex use-cases.

### Libraries

ComposeDB exposes two complementary development libraries: [`@composedb/devtools`](./api/modules/devtools.md) for generic interactions with composites, and [`@composedb/devtools-node`](./api/modules/devtools_node.md) with additional functions to interact with the local file system and start a local HTTP server.

  </TabItem>
